# ToDoList-Films

Ce projet réalisé dans le cadre de mes études me permet de travailler à la fois sur le front avec REACT et à la fois sur le back avec l'utilisation d'une API. 

##Installation 
1- Télécharger le dossier du projet
2- Ouvrir le terminal
3- Aller dans le dossier API et installer les dépendances avec 'npm install'
